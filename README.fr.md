# Brice Doublet — Portfolio

Un site portfolio moderne, immersif et interactif présentant l'expertise technique de Brice Doublet, Ingénieur DevOps & Développeur Full-Stack. 
Développé avec React, Vite et Three.js pour des performances maximales et un impact visuel fort.

## Fonctionnalités Principales

- **Stack Technique Moderne** : Construit avec React 18 et Vite pour des performances optimales et un rechargement à chaud (HMR).
- **Expériences 3D** : Logos et environnements 3D interactifs générés via Three.js (Ansible, Terraform, Full-Stack).
- **Workflows Interactifs** : Simulations de terminaux en direct et diagrammes d'architecture sur Canvas démontrant les pipelines DevOps et les infrastructures Cloud.
- **Entièrement Responsive** : Design system "mobile-first" utilisant des modules CSS et des variables personnalisées.
- **Interface Glassmorphism** : Esthétique moderne avec des panneaux translucides et des couleurs d'accentuation dynamiques par section.
- **Performances Optimisées** : Code-splitting (chargement différé), optimisation des ressources et scores Lighthouse élevés.

## Technologies

- **Cœur** : React, React Router
- **Build Tool** : Vite
- **3D & Canvas** : Three.js, API Canvas native
- **Animations** : GSAP (GreenSock), Intersection Observer API
- **Style** : CSS Modules, CSS natif avec variables

## Développement Local

Pour lancer ce projet localement :

1. **Cloner le dépôt :**
   ```bash
   git clone <votre-url-de-depot>
   cd portfolio
   ```

2. **Installer les dépendances :**
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement :**
   ```bash
   npm run dev
   ```

4. **Compiler pour la production :**
   ```bash
   npm run build
   ```
