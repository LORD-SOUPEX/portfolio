# Brice Doublet — Portfolio

A modern, immersive, and interactive portfolio website showcasing the technical expertise of Brice Doublet, DevOps Engineer & Full-Stack Developer. 
Built with React, Vite, and Three.js for maximum performance and visual impact.

## Key Features

- **Modern Tech Stack**: Built with React 18 and Vite for blazing fast performance and HMR.
- **3D Experiences**: Interactive 3D logos and scenes powered by Three.js (Ansible, Terraform, Full-Stack environments).
- **Interactive Workflows**: Live terminal simulations and architecture canvas diagrams demonstrating DevOps pipelines and Cloud infrastructure.
- **Fully Responsive**: Mobile-first design system using CSS Modules and custom variables.
- **Glassmorphism UI**: Modern aesthetic with translucent panels and dynamic accent colors per section.
- **Optimized Performance**: Code-splitting, lazy-loading, and optimized asset delivery for high Lighthouse scores.

## Technologies

- **Core**: React, React Router
- **Build Tool**: Vite
- **3D & Canvas**: Three.js, React Three Fiber (conceptually), native Canvas API
- **Animations**: GSAP (GreenSock), Intersection Observer API
- **Styling**: CSS Modules, Vanilla CSS with custom properties

## Local Development

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
